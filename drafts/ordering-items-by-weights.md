# Ordering Items By Weights

Something I like doing in my spare time is creating lists, I have come up with a list system where you order items by different weighted categories. It goes something like this:

**Score** = s1^n + s2^n + s3^n …

Where s is the individual categorical score, and the power n is the term number. So in practice let us say I was ranking programming languages on the following categories which I prioritized as matching my tastes:

Ease, Structure, Ecosystem, Procedure over Abstraction, Performance
